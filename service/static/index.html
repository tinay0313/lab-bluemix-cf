Pet Demo REST API Service

GET /api/pets to see all the pets.

GET /api/pets?category=dog to see all of the pets in the 'dog' category.

GET /api/pets/{id} retrieves a pet from the DB using an ID.

PUT /api/pets/{id} updates a pet using an ID.

DELETE /api/pets/{id} removes a pet from the DB using an ID.

POST /api/pets creates an new pet.

Example Data:

    {
        "name": "bowser",
        "category": "dog"
    }
